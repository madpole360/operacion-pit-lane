🔍 Qué es Operación Pit-Lane

Operación Pit-Lane es un proyecto independiente de periodismo de datos que investiga el coste real del Gran Premio de Fórmula 1 de Madrid. No tiene afiliación política, institucional ni mediática. No recibe financiación de ningún organismo público ni privado.

El proyecto nace de una constatación: el discurso oficial dice que "no hay dinero público", pero los expedientes de contratación cuentan otra historia. La opacidad en torno al canon de la FOM, la dispersión de contratos entre IFEMA, Comunidad y Ayuntamiento, y la experiencia del circuito de Valencia hacen necesario un seguimiento continuo y verificable.

Todos los datos publicados proceden de fuentes oficiales: la Plataforma de Contratación del Sector Público, el portal de licitaciones de IFEMA, el BOE, el BOCM, y los portales de transparencia de las administraciones implicadas. Las estimaciones y proyecciones están siempre identificadas como tales.

📐 Principios

  • Verificabilidad: cada cifra confirmada tiene al menos una fuente oficial enlazada. Si una cifra no puede verificarse, se indica expresamente.
  • Diferenciación: nunca se mezclan conceptos. Se distingue entre presupuesto base, licitación, adjudicación, modificación y gasto ejecutado.
  • Transparencia: todo el código que genera esta página es público y auditable.
  • Independencia: el proyecto no acepta patrocinios ni publicidad. Se financia exclusivamente con propinas voluntarias a través de Ko-fi.
  • Actualización diaria: el agente se ejecuta automáticamente cada noche a las 21:00h (hora Madrid) para incorporar nuevas publicaciones oficiales.

⚙️ Cómo funciona técnicamente

⏰ GitHub Actions

Un workflow programado se ejecuta automáticamente de forma periódica. Es gratuito para repositorios públicos y no requiere ningún servidor.

🔍 Monitor de portales

El Paso 0 escanea automáticamente licitaciones2.ifema.es, contrataciondelestado.es, transparencia.madrid.es y comunidad.madrid buscando nuevos expedientes con los términos "madring", "fórmula 1", "gran premio" y "circuito".

🤖 Haiku 4.5 + Web Search

El Paso 1 utiliza claude-haiku-4-5 con búsqueda web nativa para recopilar datos de fuentes oficiales, portales de contratación y medios contrastados. Es 3× más barato que Sonnet.

🧠 Sonnet 4.6 + Análisis

El Paso 2 aplica la metodología rigurosa: clasifica cada dato en 🟢🟡🟠🔴, verifica fuentes, distingue las 4 cifras de coste y genera el informe JSON. Sin web search (ya no lo necesita). Si el monitor no detecta novedades, este paso se salta para ahorrar costes.

🐍 Python + Jinja2

Un script en Python procesa la respuesta, valida los datos contra el formato real de expedientes IFEMA (NN/NNN), actualiza la base de contratos histórica sin duplicados, y genera las páginas HTML con Jinja2.

🌐 GitHub Pages

El sitio se sirve desde GitHub Pages, sin coste de alojamiento. Las páginas se regeneran automáticamente tras cada investigación.

📊 GoatCounter

Analítica web anónima, sin cookies, sin recogida de datos personales y sin necesidad de banner de consentimiento.

☕ Ko-fi

El proyecto se financia mediante propinas voluntarias. Sin publicidad, sin patrocinadores, sin suscripciones.

⚠️ Limitaciones

Lo que este proyecto NO es

  • Una fuente de asesoramiento financiero o legal.
  • Una plataforma de denuncia o activismo.
  • Un medio de comunicación con redacción humana.
  • Un proyecto afiliado a ningún partido político.

Lo que este proyecto SÍ es

  • Un registro cronológico de contratos públicos.
  • Una herramienta de consulta para periodistas e investigadores.
  • Un experimento de periodismo automatizado con IA.
  • Un proyecto transparente con código abierto.
